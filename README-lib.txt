Description
===========

This library includes useful utility functions for use in implementing
GENI AM API compatible services.


Software Dependencies
=====================
On RPM based systems:

python 2.6
m2crypto	
xmlsec1-openssl-devel
libxslt-python
python-ZSI
python-lxml
python-setuptools
python-dateutil


Installation
============
python setup.py install
